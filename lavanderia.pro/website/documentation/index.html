ProLaundry HTML

Premium HTML 5 website

  • Created: September 2020.

ProLaundry HTML is Html5 + CSS3 Website template and works fine in all major browsers. It's powered by jQuery and you'll find nice and discreet interactivity. The code is clearly written and you will find comments for each considerable parts. Build system - "Gulp".

Lets take a closer look at the structure of Html, Css, JavaScript.

Who does not want to use gulp assembly can be edited without it. You only need to use the "build" folder.
