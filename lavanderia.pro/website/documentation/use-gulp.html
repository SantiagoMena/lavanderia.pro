USE "GULP"

1)To use "GULP" you need to make sure NPM is installed. If not installed, you need to download and install:
https://nodejs.org/ru/
2)Then, in the command line(*cmd), you need to open the site and run the command "npm i"
3)After installation, you can use commands like this:
  • 1. The main commands are in the file - gulpfile.js
  • 1.2. To deploy a server - "gulp watch"
  • 1.3. To generate the production version - "gulp build"
  • 1.4. There is a command for css optimization - "gulp cssMin"
  • 1.5. There is a command for js optimization - "gulp jsMin"
  • 1.6. There is a command for html optimization - "gulp htmlMin"
  • 2.6. There is a command for all optimization - "gulp minAll"

All documentation for GULP:
https://gulpjs.com/docs/en/getting-started/quick-start
